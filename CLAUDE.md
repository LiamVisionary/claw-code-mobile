# Project context for the agent

This is **Claw Code Mobile** — a React Native (Expo) app + Express backend + bundled Rust CLI (`claw`). High-level shape lives in [ARCHITECTURE.md](ARCHITECTURE.md).

## Required reading before non-trivial work

Before writing or refactoring code in this repo, consult these and follow them:

- [DESIGN_GUIDELINES.md](DESIGN_GUIDELINES.md) — visual and interaction principles. Apply to any UI change.
- [DEVELOPMENT_GUIDELINES.md](DEVELOPMENT_GUIDELINES.md) — engineering principles (simplicity, friction, persistence, correctness).

For trivial edits (typo fixes, single-line tweaks) you can skip the read — but if you're touching UX, adding a feature, or making an architectural choice, read both first and let them shape the answer.
